STELLR

Boutique de Vêtements Interstellaire

Quelle est le but de ce projet ?

Notre Projet a été créé dans le but de répondre à une demande de projet HACKATHON en 1ère Année de BTS SIO. 

L'objectif est de créer une plateforme de commerce sécurisée disposant deux plusieurs services interactifs. 

Quelles Services Interactifs ? 

 -- Un Site Web disposant d'un Frontend et d'un Backend sur des VM distinctes en communcation        via une API.
 -- Un GitHub Public et des collaborateurs disposant des droits d'accès.
 -- Un tableau de gestion de projet accessible sur l'application TRELLO
 -- Un GLPI avec des utilisateurs distinctes et disposant de droits limités afin de créer et         répondre à des tickets. 
 -- Une Interface Zabbix permettant de faire du monitoring de chaque VM.

Quelles Sécurité Mise en Place ?

 -- Les VM sont uniquements accessible par clé donné par un Administrateur et hébergé sur Azure.
 -- Les Ports accessibles sont régulés afin d'éviter les intrusions malveillantes.
